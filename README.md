# \# Distributed Food Delivery System

# 

# A distributed food delivery platform developed for the "Distributed Systems" course (Spring 2025). The system implements a MapReduce architecture with Java backend (TCP sockets) and Android client.

# 

# \## Team

# \- Karapetsa Konstantina

# \- Sotiropoulou Foteini

# 

# \## Architecture

# \- \*\*Master Server\*\*: Java TCP server, coordinates workers, MapReduce engine

# \- \*\*Worker Nodes\*\*: In-memory data storage, handle store operations

# \- \*\*Manager Console\*\*: CLI for store management

# \- \*\*Android Client\*\*: Mobile app for customers

# 

# \## Features

# \### Manager

# \- Add/remove stores

# \- Manage products

# \- View sales reports

# 

# \### Customer

# \- Search stores within 5km

# \- Filter by category, stars, price

# \- Purchase products

# \- Rate stores
